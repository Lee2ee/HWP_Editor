사용 방법

        
      

      

        
1. 파일 업로드

        
HWP 파일을 드래그하거나 파일 선택 버튼으로 업로드하세요.

        
        
2. 텍스트 추출

        
자동으로 HWP 파일에서 텍스트를 추출하여 JSON 형태로 변환합니다.

        
        
3. 편집 및 저장

        
추출된 텍스트를 에디터에서 편집하고, JSON으로 저장하거나 다운로드할 수 있습니다.

        
        
주의: 현재는 데모 버전으로 실제 HWP 파일 파싱 대신 샘플 텍스트를 표시합니다.
